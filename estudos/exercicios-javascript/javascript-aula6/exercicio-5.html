Vamos fazer uma página para criar um quiz composto de várias questões de múltipla escolha.

Existe um input para digitar o enunciado da questão e 3 inputs para as opções (cada questão terá 3 opções). Ao lado de cada opção, existe um checkbox para marcar se aquela opção é a correta ou não. Cada questão deve ter 1 e somente 1 opção correta. Se o usuário não marcar nenhuma opção correta ou marcar mais de uma como correta, o botão “Adicionar” deve exibir um erro e não adicionar a questão.
