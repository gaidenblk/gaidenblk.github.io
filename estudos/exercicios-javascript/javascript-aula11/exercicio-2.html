Implemente uma função chamada range que funciona da seguinte forma:

• Se invocada como range(《número》), por exemplo range(4), ela retorna um array que vai de 0 até o número solicitado (exclusivo). Por exemplo range(4) ⇒ [0, 1, 2, 3] Se o número for menor ou igual a zero, ela retorna [].

• Se invocada como range(《número》, 《número》), ela trata o primeiro número como início do intervalo (inclusivo) e o segundo número como fim do intervalo (exclusivo). Por exemplo range(3, 7) ⇒ [3, 4, 5, 6] Se o segundo número for menor ou igual ao primeiro, ela retorna [].

Você pode assumir que quem invoca a função passará sempre números inteiros como valores dos parâmetros.

Sugestão: pense separadamente como você faria se fosse somente range(《número》), e se fosse somente range(《número》, 《número》 ), depois pense como você pode detectar qual dos dois casos realmente é.
