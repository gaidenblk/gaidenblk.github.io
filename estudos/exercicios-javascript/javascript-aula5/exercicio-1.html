Faça uma página que tem um <select> para o usuário escolher um produto. Os produtos podem ser o que você quiser (pelo menos 3). Quando o usuário muda o valor do <select>, a página deve exibir a imagem do produto selecionado. Exemplo: mudou para “Café”, então a página deve exibir a imagem do café. Note que não tem um botão “Mostrar”, é somente o <select>, ao ser alterado ele automaticamente causa a mudança da imagem exibida.
