Faça um <input type="text"> para preenchimento do nome completo da pessoa. Esse input só deve aceitar letras (minúsculas ou maiúsculas, somente sem acentos) e espaços em branco. Se o usuário digitar um caractere que não é permitido, o caractere simplesmente não aparece no input como se não tivesse sido digitado. Sugestão: pesquise sobre event.key Sugestão 2: saiba que o evento “input” é disparado depois que o input mudou de valor.
