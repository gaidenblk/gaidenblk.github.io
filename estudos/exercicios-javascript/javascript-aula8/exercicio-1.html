Faça uma página web que tem um parágrafo, um input de texto, e dois botões escritos “push” e “pop”.

A página deve ter um array global. O botão “push”, quando clicado, adiciona no final do array global o texto do input.

E o botão “pop”, quando clicado, remove o último elemento do array (não importa o que tem no input, já que não estamos adicionando nada no array).

O parágrafo no topo da página deve sempre exibir a representação JSON do array , que deve ser atualizada automaticamente quando o array é alterado por um dos botões. As informações da página devem ser salvas no Local Storage, ou seja, ao recarregar, o array e o parágrafo devem voltar como estavam antes da página ser fechada (note que somente o array deve ser salvo, mas não o texto do parágrafo, porque o texto do parágrafo deriva do array).
