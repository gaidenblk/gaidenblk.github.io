Faça uma aplicação web onde o usuário pode digitar o nome de uma cidade (<input type="text">) e clicar em “Buscar”.

Se houver uma filial na cidade buscada, a aplicação exibe o endereço, telefone e gerente dessa filial.

Se não houver uma filial na cidade, a página informa “Não há lojas na cidade buscada”. Sua aplicação deve ter o array acima como uma variável global (simulando o banco de dados de filiais da loja).

Use o array de exemplo para testar sua aplicação, mas ela não deve depender especificamente dos dados que estão nesse exemplo, ou seja, se trocarmos o array por outro, a aplicação deve funcionar também.
