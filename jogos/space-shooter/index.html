🚀 Space Shooter - Projeto de Aprendizado

Este projeto é um jogo do gênero space shooter desenvolvido como parte dos meus estudos com o GameMaker Studio 2, seguindo o curso do NoNe.

🎯 Objetivo

O principal objetivo deste projeto é praticar conceitos fundamentais de desenvolvimento de jogos, incluindo:

  • Movimentação de objetos
  • Sistema de tiros
  • Colisões
  • Gerenciamento de inimigos
  • Lógica de pontuação
  • Organização de código no GameMaker

🕹️ Sobre o Jogo

No Space Shooter, o jogador controla uma nave espacial e deve enfrentar ondas de inimigos, desviando de ataques e eliminando ameaças para alcançar a maior pontuação possível.

🛠️ Tecnologias Utilizadas

  • GameMaker Studio 2
  • GML (GameMaker Language)

📚 Aprendizados

Durante o desenvolvimento deste projeto, estou explorando:

  • Estrutura de projetos no GameMaker
  • Boas práticas de organização
  • Lógica de gameplay
  • Criação de mecânicas simples e funcionais

🚧 Status do Projeto

Em desenvolvimento — focado no aprendizado e evolução contínua.

💡 Observações

Este projeto faz parte da minha jornada de aprendizado em desenvolvimento de jogos. Melhorias e novas funcionalidades podem ser adicionadas conforme avanço nos estudos.
