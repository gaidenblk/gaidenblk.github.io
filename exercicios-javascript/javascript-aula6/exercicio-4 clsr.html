Versão Closure

Vamos fazer uma lista de tarefas com prioridade. A página terá um <input> para digitar a tarefa, e um botão “Adicionar” para adicioná-la. Ao ser adicionada, a tarefa aparece numa lista de tarefas (pode ser uma <ul> onde cada tarefa é uma <li>, ou como preferir). A princípio, todas as tarefas têm prioridade baixa. Ao clicar no texto de uma tarefa que já foi adicionada, ela muda para prioridade média, e isso é visualmente representado fazendo o plano de fundo da tarefa ficar amarelo. Ao clicar de novo no texto da tarefa, ela muda para prioridade alta, representada com fundo vermelho. Ao clicar de novo na tarefa, ela volta para prioridade baixa (sem cor de fundo). E assim por diante de maneira cíclica: se clicar de novo volta para prioridade média (etc.)

Evidentemente, cada tarefa tem sua prioridade independente das outras tarefas.
