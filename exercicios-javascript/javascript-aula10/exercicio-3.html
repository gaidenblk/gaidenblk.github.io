Partindo da solução final da aplicação de Lista de Filmes resolvida no material desta aula, vamos adicionar mais uma funcionalidade a ela:

Agora a tabela de filmes deve exibir também o orçamento do filme (propriedade “budget”). Note que alguns filmes estão com budget faltando, eles devem ser jogados fora no momento da importação.

A funcionalidade de filtro deve permitir filtrar por orçamento (mínimo e máximo).

Se o usuário deixar vazio o campo de mínimo, significa que não há um mínimo desejado (analogamente para o campo de máximo).

E a funcionalidade de ordenação deve permitir ordenar por orçamento (crescente ou decrescente).

Encontramos no site Kaggle uma base de dados sobre filmes, contendo informação sobre ~5000 filmes (título, gêneros, ano, orçamento, etc.): https://www.kaggle.com/datasets/carolzhangdc/imdb-5000-movie-dataset
