O objetivo deste exercício é observar como funciona a ordem de execução do código pelo console

A estrutura co código se encontra da seguinte forma:

  • console.log(1)
  • console.log(2)
    • chamada da função do botão que chama esses dois logs
    • console.log(3)
    • console.log(4)
  • console.log(5)

Resultado esperado:

  • console.log(1)
  • console.log(2)
  • console.log(5)
  • Logs que vierem depois de clicar no botão
  • console.log(3)
  • console.log(4)
