Vamos fazer uma página de anotação de lembretes.

A página terá duas seções: Na primeira seção há dois inputs, um para anotar o título do lembrete, outro para anotar a data (<input type="date">) , e um botão “Adicionar”.

Ao clicar no botão adicionar, a página não deve mostrar o lembrete, mas sim adicioná-lo no final de um array global que armazena todos os lembretes que já foram criados. Os lembretes devem ser salvos no Local Storage para que não sejam perdidos mesmo que a página recarregue.

Na outra seção da página, há um input type=“date” e um botão “Mostrar lembretes”. Ao clicar nesse botão, todos os lembretes cuja data corresponde à data escolhida são exibidos dentro de uma <ul> (cada lembrete é exibido como uma <li> dentro dessa <ul>).

Se o usuário mudar o dia e clicar de novo no botão, o conteúdo da <ul> deve ser apagado e substituído pelos lembretes do novo dia escolhido.
